# مؤسسة المنى - نظام المحاسبة المتكامل

## وصف المشروع
نظام إدارة محاسبة شامل لمؤسسة المنى يدعم:
- تسجيل وإدارة المعاملات المالية
- إدارة الرواتب والموظفين
- التفقد الأسبوعي وتتبع الحضور
- قاعدة تخزين البيانات والأرشيفات
- إنشاء تقارير PDF

## الملفات المرفوعة
- `favicon.svg` - شعار التطبيق
- `index.html` - الصفحة الرئيسية
- `robots.txt` - تعليمات محركات البحث
- `google-apps-script.js` - البرنامج النصي لـ Google Apps Script

## التشغيل
1. استنسخ المستودع
2. افتح `index.html` في متصفح الويب
3. قم بإعداد رابط Web App من Google Apps Script

## المميزات
- واجهة مستخدم عربية حديثة
- تصميم مستجيب وسريع
- دعم كامل للأجهزة المحمولة
- حفظ البيانات محليًا
- مشاركة التقارير عبر WhatsApp
