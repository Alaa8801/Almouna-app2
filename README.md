# نظام مؤسسة المنى المحاسبي

## Almouna Accounting System

تطبيق ويب متكامل لإدارة المحاسبة والموارد البشرية.

### المميزات

- 📝 **تسجيل المعاملات**: تتبع شامل للمعاملات المالية
- 💰 **إدارة الرواتب**: حساب ومراقبة رواتب الموظفين
- 📊 **التفقد الأسبوعي**: نظام متقدم لتتبع الحضور والعمل الإضافي
- 🏢 **قاعدة التخزين**: إدارة السجلات والأرشيفات
- 👥 **إدارة الموظفين**: قاعدة بيانات الموظفين الكاملة
- 🧮 **الحاسبة المدمجة**: حاسبة متقدمة
- ⚙️ **الإعدادات**: تخصيص كامل للتطبيق

### التثبيت

1. استنسخ المستودع
2. افتح `index.html` في المتصفح
3. أضف رابط Web App من Google Apps Script في الإعدادات

### التكنولوجيا

- React 19.1.0
- JavaScript (ES6+)
- CSS3
- Google Sheets API
- IndexedDB
